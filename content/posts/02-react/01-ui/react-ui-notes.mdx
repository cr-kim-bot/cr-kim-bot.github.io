## Clone이 아니라 Reference UI

Reference UI는 복제 대상이 아닙니다. 색, 간격, 배지, 목록 밀도 같은 시각 언어를 빌려오되, Developer Blog에는 읽기 흐름만 남깁니다.

## Post Hub

Post Hub는 landing page가 아니라 독자가 Post로 들어가는 시작점입니다.

### 핵심 구성

- featured Post
- recent Posts
- Category Browsing

## 가벼운 인터랙션

Theme toggle처럼 읽기에 도움 되는 조작만 둡니다. 작성 화면, auth, CMS는 MVP 범위 밖입니다.
